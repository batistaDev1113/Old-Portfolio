Memory Game

This is a game built with Javascript, CSS, and HTML. The user will have the opportunity to test their memory by matching two cards. A timer will be activated once the user clicks on a card the first time, and there is a rating functionality that will remove stars when there have been too many moves. Good luck!

Keyboard Tester

In this project, I used HTML, CSS, and Javascript. The idea behind this project is for users who are interested in knowing the keycode for every key on their keyboard. Once the user presses down any key, the value will be displayed on the screen. Enjoy it!

Pixel Art Maker

The pixel maker is a project where a board will be dynamically drawn on the screen depending on the size of rows and columns chosen by the user. There is also a color picker functionality. This little game was built using HTML, CSS, and Javascript. I hope you like it as much as I do!
