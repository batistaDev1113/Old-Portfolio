Movie Finder

This application was built using the React framework. Data is being fetch from the TMDB API and using the React Hooks. The user will be displayed with the twenty most recent movies, and they will be able to search for specific movies as well. Once the image is hovered, the overview of the movie will be displayed. The title and average rating is also displayed on the card. I hope you like movies

RoboFriends

RoboFriends is a project built using the javascript framework react, where the user will be populated with some robots from the robohash API. User will be able to search for a specific robot by name or email. The app has been created using react hooks. If you are like me, then you like robots!

Memory Game

This is a game built with Javascript, CSS, and HTML. The user will have the opportunity to test their memory by matching two cards. A timer will be activated once the user clicks on a card the first time, and there is a rating functionality that will remove stars when there have been too many moves. Good luck!

See it live Github

Neighborhood Map

This application was built using React( a javascript library ), the Google maps API and a third party API like Fourthsquare in order to render images of the resorts. Here the user can finder some locations for Disney World Resorts in the Orlando, FL area. There is also a built in search bar menu for the user to input an specific resort name. I hope you find what you are looking for!

See it live Github

My Reads Bookshelf

My Reads is a project built using React ( a javascript library) where the user can search for books by entering an author's name, of simply the name of the book. A list of books will be displayed and the reader will have options to select it as either read, currently reading or want to read. It is reading time now!
